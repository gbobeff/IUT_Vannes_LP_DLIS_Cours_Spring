Exemple : Pages HTML (statiques et dynamiques)

  • Utilisation du moteur de rendu Thymeleaf pour les pages dynamiques
    • Thymeleaf est un template engine (moteur de rendu de document) écrit en Java. Principalement conçu pour produire des vues Web
    • il permet de créer des templates qui respectent le format du document. Si vous écrivez un template pour une page HTML, le template sera une page HTML valide
    • on peut concevoir des templates pour un site Web avec des données d’exemple et le site reste navigable hors ligne comme un simple site statique
    • il s’intègre parfaitement dans une application Spring puisque vous pourrez utiliser le langage d’expression de Spring (SpEL) pour dynamiser vos templates
